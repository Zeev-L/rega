# רגע 🌅

מלווה יומי שקט לחיזוק הביטחון העצמי וההצלחה **בעבודה**.
אפליקציית menu-bar ל-macOS: מזכירה לך במהלך יום העבודה לעצור לרגע —
לקרוא את החזון האישי שלך, מנטרה, התכונות שלך, ולהודות — ושולחת לך בסוף היום
מייל עם כל מה שהודית עליו.

> "גם ביום קשה — זה מה שהיה טוב." המייל מגיע מעצמו ומרים את האנרגיה.

## הרעיון בקצרה

חמישה סוגי "רגעים", כל אחד נדלק/נכבה ומתוזמן בנפרד (לא הכול, ולא כל יום):

| סקשן | מה זה |
|---|---|
| **חזון אישי** | הצהרות: מה אני נותן לעבודה · מה זה גורם לי · איך אני מרגיש |
| **מנטרה** | משפט קצר שחוזר (אפשר כמה, מתחלפות) |
| **התכונות שלי** | הכוחות שאני מביא לעבודה (אפשר להבליט חלק בזהב) |
| **הודיה** | "על מה בעבודה אני אסיר תודה?" — נשמר ונשלח בערב |
| **עוגן ויזואלי** | תמונה שמחזקת + כיתוב חופשי |

לכל רגע: **שעה שאתה בוחר**, אפשרות **נודניק עד 3 פעמים**, ואז חובה לסמן שקראת.

## הרצה (פיתוח)

```bash
npm install
npm run icons     # מייצר את האייקונים
npm start
```

האפליקציה יושבת ב-menu-bar (אין אייקון ב-Dock). לחיצה על האייקון פותחת תפריט:
הגדרות ותוכן · הצג רגע עכשיו · שלח סיכום למייל · יציאה.

## מייל אוטומטי בסוף היום

השליחה עוברת דרך **Google Apps Script** קטן שרץ בחשבון Google שלך — אפס עלות, בלי שרת.
הגדרה חד-פעמית (5 דקות): ראה [`apps-script/README.md`](apps-script/README.md).
אחר כך: הגדרות → חיבור מייל → הדבק את ה-URL והסוד → "בדוק שליחה".

## אריזה להפצה (.dmg)

```bash
npm run dist
```

נוצר `.dmg` בתיקיית `dist/`. אפשר לשלוח לאנשים — כל אחד מתקין אצלו,
והנתונים שלו **מקומיים ועצמאיים** לגמרי (אין שרת משותף, אין תלות בך).

הערה: אם ה-`.dmg` לא חתום (בלי Apple Developer account), בפתיחה הראשונה
צריך **קליק-ימני → Open** פעם אחת כדי לעקוף את אזהרת Gatekeeper.

## איפה נשמרים הנתונים

הכול מקומי, בקובץ JSON:
`~/Library/Application Support/רגע/rega-data.json`
(תמונת העוגן נשמרת ליד, בתיקיית `images/`).

## מבנה

```
main.js            תהליך ראשי — tray, תזמון, חלונות, שליחת מייל
preload.js         גשר IPC מאובטח לחלונות
store.js           אחסון JSON מקומי + ברירות מחדל
windows/moment.*   מסך "הרגע" המלא (5 הסקשנים)
windows/settings.* הגדרות + עריכת תוכן
shared/styles.css  שפת העיצוב (שקיעה/זהב/RTL)
apps-script/       ה-relay למייל + הוראות הפעלה
generate-icon.js   מחולל אייקונים (בלי תלויות)
```

## מפת דרכים (הרחבות עתידיות)

- ריבוי משתמשים + סנכרון בין מכשירים (מעבר לחשבונות/ענן)
- גרסת Windows (אותו קוד Electron)
- סטטיסטיקות הודיה לאורך זמן
