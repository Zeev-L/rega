# חיבור מייל אוטומטי — הגדרה חד-פעמית (5 דקות)

האפליקציה "רגע" שולחת את סיכום היום למייל דרך **Google Apps Script** — endpoint קטן
שרץ בחשבון Google שלך ושולח את המייל דרך ה-Gmail שלך. אפס עלות, בלי שרת נפרד.

## שלבים

1. פתח **script.google.com** → **New project** (פרויקט חדש).
2. מחק את הקוד שיש שם, והדבק את כל התוכן של `Code.gs` מהתיקייה הזו.
3. בשורה `const SECRET = '...'` — החלף את הטקסט ב**מחרוזת סודית ארוכה** משלך
   (למשל צירוף אקראי של אותיות ומספרים). זכור אותה — תדביק אותה גם באפליקציה.
4. שמור (אייקון הדיסקט).
5. לחץ **Deploy → New deployment**.
   - ליד "Select type" בחר **Web app**.
   - **Execute as:** `Me` (אתה).
   - **Who has access:** `Anyone`.
   - **Deploy** → אשר את ההרשאות (Google ישאל אם לתת גישה ל-Gmail — אשר).
6. העתק את ה-**Web app URL** (מסתיים ב-`/exec`).

## חיבור באפליקציה

פתח את "רגע" → **הגדרות → חיבור מייל (Apps Script)**:
- **Web App URL** — הדבק את הכתובת מסעיף 6.
- **סוד משותף** — הדבק את אותה מחרוזת מסעיף 3 (חייב להיות זהה).
- לחץ **בדוק שליחה עכשיו** — אמור להגיע מייל.

## טוב לדעת
- המיילים נשלחים **מחשבון Google שלך**, ומגיעים ממנו.
- מגבלת שליחה של Gmail רגיל: ~מאות ביום — יותר מספיק.
- אם תרצה לשתף את האפליקציה עם אחרים — כל אחד יכול לפרוס Apps Script משלו
  (עצמאי לגמרי), או להשתמש באותו endpoint אם תרצה לרכז (עד המגבלה).
- ה"סוד" מונע ממישהו אחר שמכיר את ה-URL לשלוח דרכך מיילים.
